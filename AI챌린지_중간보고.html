04
예상과 달랐던 점 또는 어려움
⚠️
LLM JSON 출력 불안정
Claude API가 가끔 응답에 백틱(```) 마크다운 코드블록을 포함하거나 JSON 구문 오류를 발생시켜 자동화 파이프라인이 중단되는 경우가 있었습니다. 프롬프트 엔지니어링으로 빈도를 줄였지만 근본적 해결을 위해 Tool Use(Function Calling) 도입을 검토 중입니다.
🔢
숫자 카운팅 오류
LLM에게 게시글 수 집계를 맡기면 실제 데이터 수와 다르게 출력되는 경우가 발생했습니다. LLM은 카운팅보다 언어 이해에 특화되어 있어, 집계는 코드(Make.com)가 처리하고 분석만 AI에 위임하는 방식으로 전환했습니다.
💸
반복 테스트 시 토큰 비용 누적
코드 한 줄을 수정할 때마다 전체 파이프라인을 실행해야 하는 구조 특성상, Apify 크롤링과 Claude API 호출이 매번 반복되어 테스트 단계에서 예상보다 많은 비용이 발생했습니다. 테스트용 샘플 데이터 고정, 단위 테스트 분리 등의 방식으로 개선했습니다.
